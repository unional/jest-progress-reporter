Test progress reporter for jest.

## Usage

To use `jest-progress-reporter`,
add it to the `reporters` section of the Jest configuration:

```js
{
  "jest": {
    "reporters": [
      "default", // using default reporter
      "jest-progress-reporter"
    ]
  }
}
```
